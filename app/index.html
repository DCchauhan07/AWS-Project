About This Project

This project demonstrates a production-style static website deployment pipeline using Infrastructure as Code and CI/CD automation.

Key Features

  • Private S3 bucket for secure static hosting
  • CloudFront CDN with HTTPS
  • Terraform-managed infrastructure
  • Jenkins-based CI/CD pipeline
  • CloudWatch monitoring
  • S3 versioning for safer deployments

Technology Stack

AWS S3, CloudFront, IAM, CloudWatch, Terraform, Jenkins, HTML, CSS

Deployed as a cloud and DevOps portfolio project.
